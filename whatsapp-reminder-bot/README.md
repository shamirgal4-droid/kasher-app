# 🤖 בוט תזכורות אישי לוואטסאפ

בוט קטן שרץ על מחשב שלך ומנהל לך **תזכורות, רשימת משימות ומטלות** ישירות בוואטסאפ —
בצ׳אט "הודעה לעצמי" (Message Yourself). שולח לך פינג כשמגיע הזמן.

---

## ⚠️ קרא לפני שמתחילים — האם זה יכול לחסום לי את הוואטסאפ?

**תשובה כנה: יש סיכון קטן, לא אפס.**

הבוט מתחבר דרך **WhatsApp Web** (סריקת QR, בדיוק כמו וואטסאפ במחשב). שימוש בספריות
אוטומציה כאלה (whatsapp-web.js) הוא **לא רשמי ונוגד את תנאי השימוש של וואטסאפ**.
וואטסאפ *יכולה* לחסום מספרים על אוטומציה — בעיקר על **שליחה המונית / ספאם**.

בשימוש שקט ואישי כמו כאן (הודעות לעצמך, כמה ביום) הסיכון **נמוך**, אבל שתדע שהוא קיים.
כדי לצמצם אותו למינימום:

- 🔹 **הכי בטוח: השתמש במספר משני** (SIM ישן / eSIM / מספר וירטואלי), לא במספר האישי החשוב.
  ככה גם אם ייחסם — לא איבדת כלום.
- 🔹 אל תשלח דרכו הודעות המוניות ואל תפיץ ספאם.
- 🔹 השאר את המחשב והבוט דלוקים ברצף (חיבור/ניתוק תכוף נראה חשוד).
- 🔹 קצב אנושי — תזכורות אמיתיות שלך, לא מאות הודעות.

> אם המספר האישי שלך יקר לך (הרבה קבוצות, גיבוי וכו') — **מומלץ בחום להריץ על מספר משני.**

## 📦 התקנה (חד-פעמי)

צריך **Node.js 18+** מותקן על המחשב שדולק 24/7.

```bash
cd whatsapp-reminder-bot
npm install
```

בהתקנה הראשונה whatsapp-web.js מוריד דפדפן Chromium (לוקח כמה דקות). זה תקין.

## ▶️ הרצה

```bash
npm start
```

בפעם הראשונה יופיע **קוד QR בטרמינל**. בטלפון:
פתח וואטסאפ → **הגדרות → מכשירים מקושרים → קישור מכשיר** → סרוק את הקוד.

מהרגע הזה הבוט מחובר. ההתחברות נשמרת — בהפעלות הבאות לא צריך לסרוק שוב.

> טיפ: כדי שירוץ תמיד, הפעל אותו עם `pm2` (מנהל תהליכים):
> `npm install -g pm2 && pm2 start src/index.js --name reminders && pm2 save`

## 💬 איך משתמשים

שלח לעצמך בצ׳אט **"הודעה לעצמי"** את המילה `עזרה` ותקבל את כל הפקודות. בקצרה:

### משימות
| פקודה | מה עושה |
|---|---|
| `משימה לקנות חלב` | מוסיף משימה |
| `משימות` | מציג את הרשימה |
| `בוצע 2` | מסמן משימה 2 כבוצעה |
| `מחק 3` | מוחק משימה 3 |
| `נקה` | מוחק הכל |

### תזכורות
| פקודה | מה עושה |
|---|---|
| `תזכיר בעוד 10 דקות לשתות מים` | תזכורת בעוד 10 דקות |
| `תזכיר בעוד 2 שעות פגישה` | תזכורת בעוד שעתיים |
| `תזכיר היום ב-18:00 חדר כושר` | תזכורת להיום |
| `תזכיר מחר ב-9:00 להתקשר לרופא` | תזכורת למחר |
| `תזכיר כל יום ב-8:00 לקחת ויטמין` | תזכורת יומית חוזרת |
| `תזכורות` | מציג תזכורות פעילות |
| `בטל 1` | מבטל תזכורת מספר 1 |

## 🔧 הגדרות (אופציונלי)

משתני סביבה:

- `ALLOW_ALL_CHATS=1` — לאפשר פקודות גם מצ׳אטים פרטיים אחרים (ברירת מחדל: רק "הודעה לעצמי").
- `CHROME_PATH` — נתיב ל-Chromium מותקן אם לא רוצים שיוריד לבד.
